About Us

Our Integrated Security Firewall provides network protection for homes and small businesses using a Raspberry Pi. It combines firewall rules, DNS-based ad blocking, and an Intrusion Detection System (IDS) into one solution. The web dashboard offers real-time monitoring and easy management. It detects threats like DoS, MITM, ARP spoofing, and reconnaissance attacks, ensuring a secure and controlled network.

This Raspberry Pi-based hardware firewall is a cost-effective and scalable solution that secures networks by filtering traffic at the gateway level. It integrates multiple security features, including Intrusion Detection Systems (IDS) like Snort or Suricata, deep packet inspection, and customizable firewall rules to detect and mitigate threats such as DoS/DDoS attacks. A Flask-based web dashboard enables real-time traffic monitoring, threat alerts, and policy management. Leveraging tools like PyShark and Pandas, the system offers detailed network insights, and supports additional features like ad-blocking, VPN, and content filtering, making it ideal for home and small business use.

With increasing cyber threats like unauthorized access, malware, and DoS attacks, existing firewalls and ad-blockers lack integration for home and small business networks. This project develops a Raspberry Pi-based firewall combining DNS ad-blocking, customizable traffic filtering, IDS capabilities, and real-time monitoring. A Web Server ensures easy configuration and management.

This Raspberry Pi-based firewall system has a wide range of applications, particularly in environments where cost-effective, efficient, and customizable network security is essential. It is ideal for home networks to protect family members from harmful websites and intrusive ads, while also defending against cyber threats. In small businesses, it helps secure sensitive data, prevent unauthorized access, and monitor employee internet usage. Educational institutions can use it to manage and filter network traffic, ensuring safe and focused internet use among students. Additionally, it serves as a valuable tool for cybersecurity research and training, allowing users to study traffic behavior, implement firewall rules, and understand intrusion detection mechanisms in real-world scenarios—all through a compact, scalable, and easy-to-deploy solution.
